=== WooCommerce Sage ERP Connector ===
Author: skyverge
Tags: woocommerce
Requires at least: 3.5
Tested up to: 3.6.1
Requires WooCommerce at least: 2.0
Tested WooCommerce up to: 2.0.14

Export customer and order information to Sage ERP 100 via the Sage eBusiness Web Services API

See http://www.skyverge.com/contact/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-sage-erp-connector' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
